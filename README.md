# cub3D

## Requirements
### Requirements for Linux:
* make
* gcc
* package xorg 
* package libxext-dev 
* package libbsd-dev

➜ Requirements installation for Debian Distributions:
```
sudo apt-get install gcc make xorg libxext-dev libbsd-dev
```

### Requirements for Mac:
* Xquartz

➜ Requirements installation:
```
Brew install Xquartz
reboot
xeyes
```
